Group Evolution - API para diário de classe em MongoDB

Eu projetei uma estrutura de banco de dados NoSQL para um aplicativo web de planejamento de professores chamado "Group Evolution". O projeto se concentra em registrar atividades de aula, distinguir entre diferentes professores e facilitar a transferência de dados entre os professores.
